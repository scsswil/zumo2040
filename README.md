# Introduction to Robotics with the Zumo 2040

## What is Robotics?

Robotics is the branch of engineering and computer science that focuses on designing, building, programming, and using robots. A robot is a machine that can carry out tasks automatically, either by following instructions from a program or by reacting to its environment using sensors.

Robots are used in many different areas of life, including:

 - Manufacturing (e.g. building cars)
 - Medicine (e.g. surgical assistance)
 - Exploration (e.g. space and underwater missions)
 - Everyday technology (e.g. vacuum cleaners and delivery systems)

In this programme, you will learn how to program a small robot to behave "intelligently". You will start with simple movement and gradually build up to creating an autonomous robot that can compete in challenges.

## Meet the Zumo 2040 Robot

The Zumo 2040 is a small, tracked robot designed for learning programming, electronics, and robotics [](#zumo_2040_front). It is built around a microcontroller (a small computer on a chip) and includes motors, sensors, and outputs that allow it to move and react to its environment.

You will use the Zumo 2040 to:

 - Write programs that control movement
 - Read sensor data from the environment
 - Create autonomous behaviours
 - Compete in robot challenges such as sumo battles

The robot is small, robust, and designed for classroom and club use, making it ideal for hands-on experimentation.
